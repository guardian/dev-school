## Learning Objective

Lesson Objectives:

Getting started:

- Learn how to install Scala and SBT, and setting up your IDE
- Learn about SBT and the build.sbt file

Basic Syntax:

- Values and variables
- Types
- Comments
- Print statements
- String interpolation
- Blocks
- Collections

Conditional constructs and loops:

- If-else statements
- While loops
- For loops

Functions:

- How to write functions and methods in Scala
